# higashine-discipline-python ♨️🐍

**「最新のゴミの山」を、琥珀色の秩序へ。**

このリポジトリは、東根温泉秘密結社ボスによる、AI時代のエンジニアのための「規律（Discipline）」の実証フィールドです。

## 🏛 秘密結社の理念
「動くは最低条件、保守性こそがエンジニアの誠実さである」

AIにコードを書かせる際、人間側に「規律」がなければ、生成されるのは負債の塊です。
本プロジェクトでは、同一の要件に対して「規律なし」と「規律あり」でどのような差が生まれるかを可視化し、AIを最強のアーキテクトに変貌させる手法を提示します。

## 🧭 ディレクトリ構成
- `01_chaos_implementation/` : 規律なきAIによる「動くだけ」のスパゲッティコード
- `02_ordered_discipline/` : 秘密結社の規律を適用した「琥珀色」のレイヤー設計
- `docs/` : 設計方針および規約ドキュメント

## 🗂 エージェント・レビュー
各Pull Requestでは、秘密結社のエージェント（規約警察のルル、アーキテクトのマリ等）が、規律に基づいたコードレビューを展開しています。
