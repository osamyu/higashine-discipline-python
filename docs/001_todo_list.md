# 開発依頼：CLI Todo管理システム

以下の仕様に基づき、コマンドラインで動作するTodo管理システムをPythonで作成してください。

## 1. Todoの構成
各Todoは以下の情報を保持すること：
- 本文（内容）
- 期限（日付など）
- ステータス（「未実行」「実行済」「破棄」のいずれか）
- 完了日（「実行済」になった瞬間の日付。未実行・破棄の場合は空）

## 2. 必要な機能
コマンドラインから以下の操作を実行できるようにしてください：

- Todoの追加：本文と期限を入力して作成。初期ステータスは「未実行」。
- Todoの完了：指定したTodoのステータスを「実行済」に変更する。この際、実行した日の日付を「完了日」として自動的に記録すること。
- Todoの編集：登録済みのTodoの本文や期限の内容を更新する。
- Todoの破棄：指定したTodoのステータスを「破棄」に変更する。完了日は記録しない。
- 一覧表示：
    - デフォルトでは「未実行」のTodoのみを表示する。
    - オプション等で「実行済（完了日付き）」や「破棄」のTodoも確認できること。

## 3. 技術条件
- 言語：Python（標準ライブラリのみ、あるいは必要な最低限のライブラリ）
- 保存先：SQLite（データベースファイルを使用）
- インターフェース：コマンドライン操作（対話的、あるいは引数指定で操作できること）
